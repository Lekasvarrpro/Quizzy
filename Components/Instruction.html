Quizzy Instructions

How to Play

  • The quiz contains multiple-choice questions
  • Each question has 4 options
  • Only one option is correct
  • You cannot change your answer once selected

Time Limit

  • 20 seconds are allotted for each question
  • If the time expires, the question will be skipped automatically

Scoring

  • Correct answer: +1 point
  • Wrong answer: 0 points
  • No negative marking
